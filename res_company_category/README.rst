This module adds company categories concept.

On the company form view, you can set the category of the company.

This module is usefull for statistical purposes, in a multi company context.

Configuration
=============

* go to 'Configuration' / 'Companies' / 'Company Categories'

.. figure:: https://raw.githubusercontent.com/OCA/multi-company/12.0/res_company_category/static/description/res_company_category_tree.png

* Create new company categories.

For each category, you can define a type 'view' or 'normal' (as for product
categories).

.. figure:: https://raw.githubusercontent.com/OCA/multi-company/12.0/res_company_category/static/description/res_company_category_form.png

* Go to your companies, and set for each a category in the 'Configuration' tab.

.. figure:: https://raw.githubusercontent.com/OCA/multi-company/12.0/res_company_category/static/description/res_company_form.png
