This module allows organizations with multiples companies to transfer the
liability of a vendor bill amongst themselves.


Configuration
=============

* Go to Settings > Users & Companies > Companies
* For each company, go to the "Intercompany Payment Configuration" tab
* Set

  * the Due From Account
  * the Due To Account
  * the Due From/Due To Journal

Usage
=====

* Create Vendor Bill for the same customer in different children companies
* When Adding an Invoice Line add Distributions
* Validate invoice
